# Docker
Para executar o docker, rode os seguintes comandos:<br>
`docker network create mynet`<br>
`docker-compose up -d`

Se utiliza windows, basta rodar o <br>
`xDOCKER-START.bat`

O status da inicialização do docker é mostrado no endereço, e quando finalizado é esperado que o ambiente esteja rodando normalmente:<br>
`localhost`

Caso não tenha acesso direto aos servidores de banco de dados, conecte na VPN para que o sistema entre em operação.
